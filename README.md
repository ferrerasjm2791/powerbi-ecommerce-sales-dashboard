# E-Commerce Sales Performance Dashboard

An interactive Power BI dashboard designed to analyze profitability, sales trends, customer performance, and regional business metrics.

## Overview

This project is an interactive Power BI dashboard designed to analyze sales performance, profitability, customer behavior, and geographic trends using an e-commerce dataset.

The dashboard provides a high-level overview of key business metrics and allows users to explore sales performance through interactive filters and visualizations.

## Dashboard Pages

### 1. Profit Analysis

This page focuses on overall business performance and includes:

* Total Profit
* Total Quantity Sold
* Total Orders
* Monthly Profit Trends
* Sales Distribution by Category
* Geographic Performance Analysis
* Interactive Date Filtering

## Business Questions Answered

* How has profit changed over time?
* Which categories generate the highest sales volume?
* Which customers contribute most to profit?
* Which states perform best and worst?
* What patterns can be identified through interactive exploration?

## Key Insights

* Profit and order volume fluctuate throughout the year.
* Certain product categories contribute disproportionately to overall profitability.
* Customer profitability varies significantly across segments.
* Geographic performance highlights states with stronger and weaker profit generation.
